#Application de covoiturage inter-villes pour étudiants et travailleurs
Objectif : Mettre en relation conducteurs et passagers pour partager les trajets entre villes.
Acteurs : Conducteur, Passager, Administrateur.
Fonctionnalités minimales :
 Gestion des profils utilisateurs.
 Publication d’un trajet (ville départ, ville arrivée, heure, prix).
 Réservation de place par un passager.
 Système d’évaluations et avis.
 Notifications de confirmation.
Technologies suggérées : Servlets/JSP, MVC, MySQL, éventuellement WebSockets
pour notifications en temps réel.
